## Execution Order

1. P0 Unit (6): 默认模式、v-model 绑定、空内容检测、格式校验、大小校验、URL 必填
2. P0 Int (0)
3. P1 Unit (18): 模式切换、数据保留、提交格式、协议校验、validate 方法
4. P1 Int (7): 回显、上传交互、删除、失败重试、props 配置
5. P1 E2E (1): 完整流程
6. P1 Blind Spot (7): 边界值、网络异常、快速切换
7. P2+ (10): 大小限制、字段长度、reset、编辑流程
