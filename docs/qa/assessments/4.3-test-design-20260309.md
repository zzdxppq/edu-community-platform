| 风险 | 覆盖场景 |
|------|----------|
| XSS 注入（恶意 HTML） | WangEditor 内置过滤 + 后端 sanitize（非本 Story 范围，但配置验证） |
| 内存泄漏 | 4.3-INT-006, 4.3-BLIND-RESOURCE-001 |
| 上传安全 | 4.3-UNIT-014~016, 4.3-BLIND-BOUNDARY-004~005 |
| Token 安全 | 4.3-UNIT-017, 4.3-BLIND-ERROR-003 |

4. P1 Integration (4.3-INT-001~002, 004~007) — 组件交互
5. P1 Blind Spot (BOUNDARY-001~003, ERROR-001~003, FLOW-001~002, 004, RESOURCE-001)
6. P1 E2E (4.3-E2E-001) — 端到端验证
7. P2+ 剩余场景
