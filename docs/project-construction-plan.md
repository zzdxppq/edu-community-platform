## 第一章 项目概述

### 1.1 项目背景

为深入贯彻国家关于推进县域城乡义务教育一体化发展的战略部署，促进河南省城乡学校在师资培养、课程建设、教学教研等方面的资源共享与优势互补，亟需建设一套综合性信息服务与管理平台。该平台将整合政策文件、示范校经验、教学资源等核心信息资源，为各级教育行政部门、学校管理者、教师及社会公众提供高效、便捷的信息获取渠道，助力河南省城乡教育均衡发展。

### 1.2 项目定位

面向河南省县域城乡义务教育相关主体的**综合性开放型信息服务与管理平台**，聚焦以下四大核心场景：

| 序号 | 核心场景 | 场景说明 |
|------|---------|---------|
| 1 | 政策传达 | 及时发布国家及省级教育政策文件，提供权威政策解读 |
| 2 | 经验分享 | 展示示范校建设经验与成果，促进校际交流学习 |
| 3 | 资源共享 | 提供教学资源、研究文献等材料的在线预览与下载 |
| 4 | 示范校展示 | 各示范校拥有独立展示页面，全面呈现校共同体建设成效 |

### 1.3 服务对象

| 用户类型 | 细分群体 | 平台角色 |
|---------|---------|---------|
| 访客用户 | 教师、教研员、家长、媒体、教育行政人员等 | 浏览所有公开内容，无需注册登录 |
| 学校管理员 | 各示范校指定管理人员 | 管理本校内容（校共同体介绍、活动、月报、成员校） |
| 超级管理员 | 平台运营人员 | 全平台内容管理、用户管理、系统配置 |

### 1.4 建设范围

本项目采用**分期建设**策略：

- **一期（P0）— 网站建设**：完成平台基础搭建与后台运营闭环，包含官网前端五大核心模块、超级管理运营后台、学校管理后台、全站搜索和管理员登录系统。
- **二期（P1）— 智能体**：完善用户注册登录体系，建设知识库，上线 AI 智能体服务（基于 RAG + 通义千问），实现个性化推荐与智能问答。

## 第二章 建设目标

### 2.1 一期建设目标

| 目标维度 | 目标描述 | 量化指标 |
|---------|---------|---------|
| 网站搭建 | 完成开放型官网及运营后台建设 | 前端5大模块 + 2套后台系统上线 |
| 内容管理 | 实现平台信息的正常维护、更新 | 支持图文编辑、附件上传、外部链接三种发布方式 |
| 学校管理 | 支撑学校管理员开展本校内容管理 | 首批6所示范校接入运营 |
| 开放浏览 | 核心内容无需注册登录即可访问 | 首页加载 ≤3s，接口响应 ≤500ms |

### 2.2 二期建设目标

| 目标维度 | 目标描述 | 量化指标 |
|---------|---------|---------|
| 用户体系 | 上线普通用户注册登录功能 | 支持7种身份标签，手机号/邮箱注册 |
| 智能服务 | 嵌入 AI 智能体，提供个性化推荐与智能问答 | RAG 检索 + 通义千问大模型 |
| 知识库 | 建设平台知识库，实现内容自动采集与人工录入 | 自动采集4类内容 + FAQ/专家问答手工录入 |
| 数据分析 | 为教育行政部门提供决策支持 | 内容统计报表、使用统计仪表盘 |

### 2.3 性能目标

| 指标项 | 目标值 |
|--------|--------|
| 首页加载时间 | ≤ 3 秒（100Mbps 网络环境） |
| 常规接口响应时间 | ≤ 500ms |
| 并发用户支持 | ≥ 500 人同时在线 |
| 服务器出口带宽 | ≥ 100Mbps |
| 文件下载首字节响应 | ≤ 3 秒 |
| 并发下载支持 | ≥ 100 人同时下载 |
| 系统可用性 | ≥ 99.5% |
| 数据恢复时间（RTO） | ≤ 4 小时 |

## 第三章 建设内容与功能设计

### 3.1 总体功能架构

```
河南城乡学校共同体发展平台
│
├── 前端官网（面向公众）
│   ├── 首页（导航栏、搜索、轮播图、模块入口、友情链接、版权信息）
│   ├── 新闻资讯（首页入口访问，导航栏不显示）
│   ├── 项目介绍（单页内容：项目名称 + 富文本内容）
│   ├── 政策文件（国家政策 / 省级政策，支持置顶、下载打印）
│   ├── 示范校共体（左侧校列表导航 + 右侧Tab切换：介绍/活动/月报）
│   ├── 资源共享（省外经验 / 省内经验 / 研究文献，支持搜索筛选）
│   ├── 全站搜索（跨模块关键词搜索）
│   └── 登录入口（管理员登录链接）
│
├── 超级管理运营后台（面向运营人员）
│   ├── 工作台（统计卡片 + 快捷操作）
│   ├── 用户管理（学校管理员增删改查 + 账号管控）
│   ├── 栏目管理（一级/二级栏目配置）
│   ├── 内容管理（轮播图、新闻、政策、资源、项目介绍、月报管理）
│   ├── 示范校管理（列表、维护、越权登录）
│   └── 系统设置（基础配置、权限、敏感词、数据备份）
│
└── 学校管理后台（面向学校管理员）
    ├── 工作台（最近活动 + 月报记录）
    ├── 个人中心（修改密码、联系方式）
    ├── 栏目管理（只读查看）
    ├── 内容管理（校共同体介绍、活动管理、月报管理）
    └── 成员校管理（添加/编辑/删除/排序）
```

### 3.2 前端官网模块设计

#### 3.2.1 首页模块

| 功能组件 | 功能说明 |
|---------|---------|
| 导航栏 | 固定顶部导航："首页、项目介绍、政策文件、示范校共体、资源共享、登录"，支持二级菜单动态渲染 |
| 全站搜索 | 导航栏右侧搜索框，搜索范围覆盖政策标题、示范校名称、活动标题、资源标题 |
| 轮播图 | 展示3-5张运营配置图片，自动轮播（5秒间隔），点击跳转对应详情 |
| 模块入口 | 各模块最新5条内容（标题+发布时间），置顶内容优先，含新闻资讯入口 |
| 友情链接 | 河南省教育厅、河南大学教育学部、河南省教育信息化发展研究中心 |
| 版权信息 | 网站地址、邮箱、版权声明（不含底部快捷链接） |

#### 3.2.2 新闻资讯模块

- 导航栏不显示，仅通过首页入口访问
- 新闻列表页：标题 + 发布时间，按时间倒序，置顶优先，支持分页
- 新闻详情页：标题 + 发布时间 + 富文本正文

#### 3.2.3 项目介绍模块

- 单页展示：项目名称 + 项目内容（富文本）
- 内容由超级管理员通过后台维护

#### 3.2.4 政策文件模块

- 按"国家政策、省级政策"分类，左侧分类导航 + 右侧列表
- 列表显示标题、发布时间、所属分类，置顶优先
- 详情页展示正文（不显示文号），含发文机构信息
- 支持 PDF 下载和在线打印

#### 3.2.5 示范校共体模块

- **左侧导航**：默认显示前6所示范校（按配置排序），支持"查看全部"展开
- **右侧内容区**：Tab 切换（校共同体介绍 | 校共同体活动 | 校共同体月报）
  - **介绍 Tab**：学校基本情况、共同体组建模式、核心优势、成员校列表
  - **活动 Tab**：最新5条活动，"查看更多"进入独立列表页 → 详情页
  - **月报 Tab**：最新5条月报，"查看更多"进入独立列表页（支持年月筛选）→ 详情页
- 页面背景：支持学校实景图片（1920×400px），未上传使用默认图片

**URL 路由设计**：

| 页面 | 路由 |
|------|------|
| 学校主页 | `/schools/:id` |
| 活动列表 | `/schools/:id/activities` |
| 活动详情 | `/schools/:id/activities/:activityId` |
| 月报列表 | `/schools/:id/reports` |
| 月报详情 | `/schools/:id/reports/:reportId` |

#### 3.2.6 资源共享模块

- 按"省外经验、省内经验、研究文献"分类，左侧分类导航
- 列表显示标题、发布时间、文件大小、所属分类、来源，置顶优先
- 支持二级分类下拉筛选 + 关键词搜索
- 支持在线预览和文件下载

#### 3.2.7 全站搜索

- 搜索结果按模块分类展示：标题、所属模块、发布时间
- 支持按模块筛选、分页
- 空结果友好提示

#### 3.2.8 全局分页规范

- 默认每页10条，可选10/20/50/100
- 显示总条数，包含首页/上一页/页码/下一页/末页/跳转

### 3.3 超级管理运营后台设计

#### 3.3.1 工作台

| 组件 | 说明 |
|------|------|
| 统计卡片 | 示范校数量、学校管理员数量、本月活动数、本月月报数 |
| 快捷操作 | 添加管理员、发布政策、发布资源、发布新闻（一键跳转） |

#### 3.3.2 用户管理

- **学校管理员列表**：支持按账号（手机号）搜索、按学校下拉筛选；新增时账号必须为手机号，默认密码为手机号后6位，不含姓名字段
- **账号管控**：禁用/启用账号，重置密码（恢复为手机号后6位并标记首次改密）

#### 3.3.3 栏目管理

- **一级栏目**：新增/隐藏/修改/删除/排序，系统预置栏目不可删除
- **二级栏目**：设置是否启用二级菜单开关，新增/编辑/删除/排序二级菜单项

#### 3.3.4 内容管理

| 功能模块 | 核心能力 |
|---------|---------|
| 轮播图管理 | 新增/编辑/删除/排序轮播图，设置图片、跳转链接、显示状态 |
| 新闻资讯 | 富文本编辑器，状态筛选（已发布/草稿），三种发布方式，置顶，无摘要字段 |
| 政策文件 | 富文本编辑器，状态筛选，三种发布方式，置顶，含发文机构字段 |
| 资源共享 | 资源描述用富文本，状态筛选，三种发布方式，置顶，超管可删全站资源 |
| 项目介绍 | 单页编辑：项目名称 + 富文本内容 |
| 月报管理 | 查看所有学校月报，按学校/时间筛选，可删除不合规月报 |

**三种内容发布方式**：

| 发布方式 | 说明 |
|---------|------|
| 图文编辑 | 使用全站统一富文本编辑器，支持图片/视频/表格/首行缩进等 |
| 附件上传 | 可选附件（PDF/Word 等），支持多文件 |
| 外部链接 | 仅填写标题和 URL，列表显示"外链"标识，点击直接跳转外部页面 |

#### 3.3.5 示范校管理

- **示范校列表**：不显示统计信息，不提供批量操作，含显示/隐藏、排序、越权登录按钮
- **示范校维护**：新增/编辑/删除示范校基础信息（名称、学段、区域、电话），设置官网显示及排序
- **越权登录**：一键登录学校管理后台（模拟登录），操作需记录日志

#### 3.3.6 系统设置

| 功能 | 说明 |
|------|------|
| 基础配置 | 网站 Logo（支持上传）、网站名称、版权信息、友情链接 |
| 权限设置 | 配置管理员权限，新增子账号 |
| 敏感词管理 | 维护敏感词库，支持批量导入/添加/删除 |
| 数据备份 | 查看备份状态，手动触发备份（每日限1次） |

### 3.4 学校管理后台设计

#### 3.4.1 工作台

- 最近发布的校共同体活动列表（分页，编辑/删除/发布按钮）
- 月报提交记录列表（分页，编辑/删除/提交按钮）

#### 3.4.2 个人中心

- 查看账号信息，修改密码、联系方式
- 首次登录强制修改密码

#### 3.4.3 内容管理

| 功能 | 说明 |
|------|------|
| 校共同体介绍 | 点击进入编辑页面，富文本编辑器维护（组建模式、核心优势等），不可修改基础信息 |

| 校共同体月报 | 列表视图管理，支持分页、编辑、删除、提交；富文本编辑器；按月份筛选 |

#### 3.4.4 成员校管理

- 查看、添加、编辑、删除成员校
- 支持拖拽排序，前端按此顺序展示
- 信息字段：学校名称、学校代码（可选）、排序

### 3.5 富文本编辑器规范

| 媒体插入 | 图片上传/粘贴、视频嵌入 |
| 表格功能 | 插入/编辑表格、合并/拆分单元格 |
| 辅助功能 | 撤销/重做、全屏编辑、源代码模式、一键清除格式 |

## 第四章 技术架构方案

### 4.1 总体架构

本平台采用 **Spring Cloud 微服务 + Vue 3 前后端分离** 架构，整体分为四层：

```
┌──────────────────────────────────────────────────────────┐
│                      接入层                               │
│   CDN (静态资源加速) + Nginx (反向代理 / HTTPS 终端)       │
└──────────────────────────────┬───────────────────────────┘
                               │
┌──────────────────────────────┴───────────────────────────┐
│                      应用层                               │
│  ┌──────────────────────────────────────────────────────┐ │
│  │                  前端应用                             │ │
│  │  官网（Nuxt 3 SSR）      管理后台（Vue 3 + Element Plus）│ │
│  └──────────────────────────────────────────────────────┘ │
│  ┌──────────────────────────────────────────────────────┐ │
│  │               API Gateway（Spring Cloud Gateway）     │ │
│  │       路由转发 / 限流 / JWT 鉴权 / 请求日志           │ │
│  └──────────────────────────────────────────────────────┘ │
│  ┌────────────┐  ┌────────────────┐  ┌────────────────┐  │
│  │core-service│  │content-service │  │ file-service   │  │
│  │认证/用户/  │  │新闻/政策/资源/ │  │文件上传下载/   │  │
│  │学校/权限   │  │活动/月报/栏目  │  │图片处理/存储   │  │
│  └────────────┘  └────────────────┘  └────────────────┘  │
└──────────────────────────────┬───────────────────────────┘
                               │
┌──────────────────────────────┴───────────────────────────┐
│                      服务治理层                            │
│     Nacos（服务注册/发现 + 配置中心）                      │
└──────────────────────────────┬───────────────────────────┘
                               │
┌──────────────────────────────┴───────────────────────────┐
│                      数据层                               │
│  MySQL 8.0（主从复制）  Redis 7（哨兵模式）  腾讯云 COS    │
└──────────────────────────────────────────────────────────┘
```

### 4.2 技术选型

#### 4.2.1 后端技术栈

| 技术组件 | 版本 | 用途 |
|---------|------|------|
| JDK | 21 LTS | 运行环境 |

### 4.4 认证授权方案

```
┌──────────┐   登录请求     ┌──────────┐   验证通过      ┌──────────┐

│          │  Refresh Token│          │               │          │
└──────────┘               └──────────┘               └──────────┘
```

- **认证模型**：JWT（Access Token 15分钟 + Refresh Token 7天）
- **授权模型**：RBAC（super_admin / school_admin 两种角色）
- **Token 管理**：Redis Token 黑名单，支持主动注销
- **密码安全**：BCrypt 加密存储，8-20位，必须包含大写、小写字母和数字
- **首次登录**：强制修改默认密码（手机号后6位）

## 第六章 安全保障方案

### 6.1 安全等级

本项目按照**等保三级**标准进行安全设计。

### 6.2 传输安全

| 措施 | 说明 |
|------|------|
| HTTPS | 全站启用 HTTPS，TLS 1.2+ |
| 证书管理 | SSL 证书自动续签 |
| HTTP 跳转 | HTTP 请求自动 301 跳转至 HTTPS |

### 6.3 认证安全

| 措施 | 说明 |
|------|------|
| 密码加密 | BCrypt 加密存储 |
| 密码策略 | 8-20位，必须包含大写字母、小写字母和数字 |
| 首次改密 | 新账号首次登录强制修改默认密码 |
| Token 机制 | JWT（Access 15min + Refresh 7天）+ Redis 黑名单 |
| 权限隔离 | RBAC 模型，学校管理员仅可操作本校数据 |

### 6.4 应用安全

| 威胁类型 | 防护措施 |
|---------|---------|
| SQL 注入 | 参数化查询（MyBatis-Plus），禁止拼接 SQL |
| XSS 攻击 | 输入过滤 + 输出转义，富文本内容白名单过滤 |
| CSRF | Token 校验机制 |
| 敏感词 | 内容发布时自动检测敏感词库，命中提示修改 |
| 外部链接 | 仅允许 http/https 协议，跳转前弹出安全提示，支持白名单 |
| 文件上传 | 类型白名单校验（jpg/png/gif/pdf/doc/docx/xls/xlsx/mp4/mov）+ 大小限制 |

### 6.5 文件安全

| 文件类型 | 支持格式 | 大小限制 |
|---------|---------|---------|
| 图片 | jpg / png / gif | 单张 ≤ 5MB |
| 文档 | pdf / doc / docx / xls / xlsx | 单文件 ≤ 50MB |
| 视频 | mp4 / mov | 单文件 ≤ 200MB |

### 6.6 操作审计

- 管理员关键操作（登录、删除、越权登录等）记录操作日志
- 日志包含：操作人、操作类型、操作对象、详情、IP、时间
- 越权登录操作必须记录日志
