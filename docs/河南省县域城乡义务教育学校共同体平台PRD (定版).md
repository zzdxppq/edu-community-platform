河南城乡学校共同体发展平台PRD
一、产品概述
（一）产品目标
P0：搭建河南城乡学校共同体发展开放型网站及运营后台，实现平台信息的正常维护、更新，支撑学校管理员开展本校内容管理工作，支持超级管理员添加、维护示范校信息。
P0：实现网站开放浏览，除智能体相关功能外，所有前端核心内容（政策、校共体信息、资源等）无需注册登录即可查看，降低访问门槛。
P1：整合政策文件、校共体经验、教学资源等核心信息，实现信息共享、经验互通，为各级教育相关主体提供高效信息获取渠道。
P1：展示示范校共体建设成果，推广优秀实践模式，发挥示范引领作用，促进全省校共体建设质量提升。
P1：完善用户体系，上线普通用户注册登录功能，为后续智能体嵌入奠定基础，实现基于用户身份的个性化服务。
（二）产品定位
面向河南省县域城乡义务教育相关主体的综合性开放型信息服务与管理平台，聚焦政策传达、经验分享、资源共享、校共体展示四大核心场景，前期优先实现网站搭建、后台运营及学校管理员管理功能，普通用户注册延后上线；平台整体开放浏览，仅使用智能体功能时需注册登录；管理后台采用统一登录页面，管理员登录时选择对应身份进入专属管理界面，超级管理员可通过后台快速访问示范校主页，提供公开、透明、高效的线上服务支撑，助力城乡教育一体化发展。
（三）使用场景
P0：信息浏览场景：所有用户（无需注册登录）浏览网站前端核心内容，包括政策文件、校共体介绍、项目进展、资源共享、新闻资讯等。
P0：后台运营场景：超级管理员通过运营后台，开展栏目管理、示范校信息添加与维护、内容发布、学校管理员账号分配/权限配置、系统设置等操作，维护平台正常运行；可直接删除共同体学校发布的不合规内容，无需审核流程；可通过后台快速访问任意示范校前端主页。
P0：学校管理场景：学校管理员通过后台，维护本校校共体介绍、校共同体活动（发布完直接到官网的新闻资讯模块）、校共同体月报，有修改和编辑权限，支持图文编辑和外部链接发布两种方式。
P1：智能体使用场景：普通用户注册登录（选择唯一身份，不可修改）后，使用智能体相关功能（个性化推荐、智能问答等）；二期智能体功能可能采用收费或内部邀请码模式，平衡大模型使用成本。
P0：登录跳转场景：用户通过官网前端登录入口进入统一管理员登录页面，页面内提供“超级运营管理员”“学校管理员”两种身份选择，选择后验证账号密码进入对应管理后台；无需区分多个登录入口，用户仅需记住官网即可完成管理员登录操作。
二、产品对象
关键规则
平台开放规则：除智能体相关功能外，所有前端核心内容（政策文件、校共体信息、资源共享等）均开放浏览，无需注册登录；仅使用智能体功能时，需完成普通用户注册登录。
普通用户规则：普通用户注册登录功能前期不开发（优先级P1），待智能体上线时同步上线；注册时需选择唯一身份（教师/教研员/学校管理者/教育研究者/家长/媒体/县级教育行政部门工作人员），无需填写单位，身份提交后不可修改，作为智能体个性化服务的核心标签；二期智能体功能因大模型使用频次高、成本高，可能采用收费或内部邀请制模式，保障平台良性发展。
权限隔离规则：学校管理员仅可操作本校相关内容栏目，无平台级管理权限、无示范校信息添加与维护权限；超级管理员拥有全平台权限，重点负责后台运营、学校管理员管理、示范校信息维护，可直接删除示范校不合规内容，无需审核流程。
发布内容规则：所有管理员发布内容时，支持图文编辑上传和外部链接发布两种方式，链接发布后点击可直接跳转至外部页面。
登录入口规则：官网前端默认是普通用户登陆，然后登陆下方有管理员登录入口，点击进入统一登录页面，页面内提供“超级运营管理员”“学校管理员”身份选择，验证账号密码后跳转至对应管理后台；用户无需记忆多个登录地址，仅需记住官网即可完成管理员登录操作。
示范校管理规则：超级管理员拥有示范校信息的添加、编辑、维护权限，可新增示范校、修改示范校基础信息；学校管理员仅可维护本校相关内容；超级管理员可通过管理后台快速访问任意示范校前端主页，便捷查看示范校展示内容。

四、功能规划模块
（一）一期规划：核心是网站建设
核心目标：完成平台基础搭建与后台运营闭环，实现前端开放浏览、后台可管控、学校管理员可操作，满足前期核心使用需求。
前端功能：上线首页、项目介绍、政策文件、示范校共体、资源共享五大核心模块，完善友情链接、版权信息展示；显示“登录”入口，登录界面下方设置“管理员登录”入口，点击跳转后台管理系统；前期隐藏注册、个人中心、智能体相关入口；所有开放内容支持浏览、预览、下载、打印。
后台功能：完成超级管理运营端核心功能（栏目配置、内容发布、学校管理员账号分配/权限配置、系统基础设置、日志管理、数据备份、学校管理员账号管控）；上线学校管理员端核心功能（本校内容发布/修改/暂存/删除、个人账号信息修改）；支持图文编辑、附件上传和外部链接发布三种内容发布方式；超级管理员可直接删除示范校不合规内容，无需审核流程。
上线标准：前端页面正常渲染、无报错，所有开放内容可正常访问；登录入口跳转正常，管理员登录可顺利进入后台；后台操作流畅，权限隔离清晰，学校管理员可正常开展本校内容管理；数据备份、日志留存功能正常生效；内容发布支持链接跳转功能。
（二）二期规划：核心是智能体
核心目标：完善用户体系、扩展功能边界、提升平台实用性与性能，实现个性化服务、功能扩展与性能优化一体化，助力校共体建设决策，满足更多使用场景需求；平衡大模型使用成本，探索可持续运营模式。
用户体系：上线普通用户注册功能，支持手机号/邮箱注册、登录、密码找回，注册时需选择唯一身份标签（不可修改，无需填写单位）；上线个人中心模块，支持个人信息修改（不含身份）、浏览历史记录、收藏管理功能。
智能体功能：嵌入智能体基础模块并完成升级，实现个性化内容推荐（基于用户身份标签）、智能问答（解答校共体相关常见问题）；新增数据可视化分析报告生成（为县级教育行政部门、高校专家提供决策支持）、个性化提醒功能（收藏内容更新、政策提醒等）；仅注册登录用户可使用智能体功能；因大模型使用频次高、成本高，智能体功能采用收费使用或内部邀请制模式，保障平台良性发展。
后台优化：升级超级管理运营端，支持普通用户列表查看、账号管控；新增内容统计报表导出功能，支持超级管理员、学校管理员导出对应权限的数据报表；优化后台操作流畅度，完善权限隔离细节。

| 智能问答 | 解答用户关于校共体政策、建设模式、资源使用等常见问题 | 仅登录用户可使用，支持关键词检索问答；二期可能采用收费或内部邀请制 | P1 |


| 功能模块 | 功能点 | 功能描述 | 优先级 |
| --- | --- | --- | --- |
| 用户管理 | 用户列表 | 前期：仅查看学校管理员账号信息、编辑和添加学校多个管理员，支持按学校名称、账号状态筛选；后期：查看所有注册用户信息，支持按身份、手机号、昵称筛选，可导出用户基础数据（不含密码） | P0（学校管理员）、P1（普通用户） |
| 用户管理 | 账号管控 | 前期：管控学校管理员账号（禁用/启用），支持重置学校管理员密码，密码重置后需通知对应管理员；后期：因大模型使用频次高、成本高，智能体功能视情况设置成收费使用或内部邀请制，可管控普通用户智能体使用权限 | P0（学校管理员）、P1（普通用户） |
| 栏目管理 | 栏目配置 | 管理前端核心栏目（项目介绍、政策文件等），支持新增/隐藏/修改/删除子栏目、调整栏目排序；重点负责示范校共体栏目的管理，可添加、编辑、维护示范校基础信息（如示范校名称、关联学校、基础介绍等），新增示范校后前端导航栏自动同步；前期重点配置开放浏览栏目，后期可配置智能体相关栏目 （栏目下已有数据不能删除只能隐藏，没数据可以删除） | P0 |
| 内容管理 | 内容发布 | 发布平台级内容（政策文件、新闻资讯、资源共享内容等），支持图文编辑、外部链接发布、附件上传，可设置发布时间（即时/定时） | P0 |
| 内容管理 | 内容管控 | 查看全平台所有内容（含学校管理员发布内容），可直接删除不合规内容、编辑错误内容，无需审核流程；支持按内容类型、发布主体、发布时间筛选 | P0 |
| 内容管理 | 内容统计 | 统计全平台内容发布数量、各栏目内容占比、学校管理员发布频次，支持导出统计报表 | P1 |
| 系统设置 | 基础配置 | 设置网站名称、版权信息、联系电话、邮箱等基础信息，支持修改后即时同步至前端；配置轮播图、友情链接，支持新增、删除、排序 | P0 |
| 系统设置 | 权限设置 | 配置超级管理员、学校管理员的权限范围，可新增超级管理员子账号（有限权限），支持权限细分（如部分学校管理员仅可发布内容，不可修改） | P1 |
| 系统设置 | 日志管理 | 记录所有管理员操作日志（登录、内容发布/删除、权限调整等），包含操作人、操作时间、操作内容，支持按条件筛选、导出，日志不可删除 | P1 |
| 示范校管理 | 示范校维护 | 新增、编辑、删除示范校信息，配置示范校关联成员校，维护示范校基础介绍（前端校共体介绍模块核心内容），支持快速跳转至示范校前端主页 | P0 |


| 功能模块 | 功能点 | 功能描述 | 优先级 |
| --- | --- | --- | --- |
| 个人中心 | 账号管理 | 查看个人账号信息（所属学校、权限范围），支持修改登录密码、联系方式，可退出登录；无权修改账号所属学校及权限 | P0 |
| 栏目管理 | 本校栏目查看 | 查看本校所属栏目（示范校共体下本校子栏目：校共体介绍、项目进展、资源成果），不可新增、隐藏、调整栏目排序，仅可查看栏目配置 | P0 |
| 内容管理 | 校共体介绍维护 | 补充、修改本校校共体介绍内容，支持图文编辑、外部链接发布、图片上传（学校环境、活动照片等），发布后即时同步至前端；不可修改超级管理员维护的示范校基础信息 | P0 |
| 内容管理 | 项目进展发布 | 发布本校校共体建设动态、阶段成果、工作推进情况，支持图文编辑、外部链接发布、视频/附件上传，发布后同步至大共同体新闻资讯板块；可修改、删除本人发布的内容 | P0 |
| 内容管理 | 校共同体月报维护 | 上传、编辑本校校共同体月报，支持图文编辑、附件上传（PDF/Word格式），可按月份筛选查看历史月报，支持修改、删除；月报内容仅展示本校相关，不影响其他学校 | P0 |
| 内容管理 | 资源成果发布 | 发布本校校共体特色课程、合作案例、获奖成果等资源，支持在线预览设置、附件上传、外部链接发布，可修改、删除本人发布的资源 | P0 |
| 内容管理 | 内容查询 | 按内容类型、发布时间、内容标题筛选本人发布的所有内容，支持快速定位修改、删除 | P1 |
| 资源管理 | 本地资源管理 | 查看本人上传的所有资源（图片、附件、视频等），支持分类筛选、下载、删除，可复用资源至其他内容发布场景 | P1 |
