## 技术决策

### 1. BCrypt 依赖选择
使用 `spring-security-crypto`（非 spring-boot-starter-security），避免安全过滤器自动配置导致 HealthCheck 等端点需要认证。

### 2. MapStruct + Lombok 联合编译
在 core-service pom.xml 的 maven-compiler-plugin 中配置 annotationProcessorPaths，lombok 必须在 mapstruct-processor 之前声明。

### 3. schoolId 校验策略
在 AdminUserMapper 中添加 `existsSchoolById` 自定义 SQL 查询 schools 表，避免提前创建 School Entity/Mapper（不在本 Story 范围）。

### 4. 软删除机制
利用 application.yml 中已配置的 MyBatis-Plus 全局逻辑删除：`deletedAt` 字段 NULL=未删除，NOW()=已删除。BaseEntity 使用 `@TableLogic(value = "NULL", delval = "NOW()")`。

### 5. 默认密码生成
手机号后6位 → BCrypt 加密。salt 字段填入 UUID 32位字符串（DDL NOT NULL 约束兼容，BCrypt 内部已含盐）。

### 6. Controller 路由
使用 `@RequestMapping("/v1/admin-users")`，Gateway StripPrefix=2 处理外部路径 `/api/core/v1/admin-users` → 内部 `/v1/admin-users`。
