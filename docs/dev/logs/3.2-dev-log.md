## 技术决策

### 1. useApiFetch 泛型设计
使用 `Record<string, unknown>` 作为 opts 参数类型，配合 `as object` 展开，避免 Nuxt 3 复杂 UseFetchOptions 泛型冲突。useBanners 中对 data.value 使用类型断言 `as ApiResult<BannerItem[]> | null` 确保 .data 属性访问安全。

### 2. CSS transition 滑动动画
使用 `transform: translateX(-${activeIndex * 100}%)` + `transition: transform 0.5s ease` 实现水平滑动，无需 JavaScript 动画库。每个 slide 宽度 100%，track flex 布局。

### 3. 箭头 hover 显示
箭头默认 opacity:0，`.carousel:hover .carousel-arrow { opacity: 1 }`。纯 CSS 实现，不需 JS 状态管理。

### 4. 占位图 CSS 渐变
使用 `linear-gradient(135deg, var(--primary-800), var(--primary-900))` 纯 CSS 渐变，不依赖外部图片文件。

### 5. 单条目处理
`banners.length === 1` 时不渲染箭头和指示器（与 banners.length > 1 条件控制），不启动定时器。
