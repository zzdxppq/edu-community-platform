## 技术决策

### 1. Flyway 版本选择
选择 Flyway 10.10.0，兼容 Spring Boot 3.2.x 和 MySQL 8.0。使用 `flyway-mysql` 扩展模块支持 MySQL 8+ 特性。

### 2. DDL 严格遵从架构文档
所有 CREATE TABLE 语句从 `docs/architecture/database-schema.md` 原样复制，未做任何字段修改。包含所有索引、外键、FULLTEXT 索引等。

### 3. Redis 序列化策略
- Key: StringRedisSerializer（字符串可读性）
- Value: GenericJackson2JsonRedisSerializer（JSON 通用序列化）
- 选择 GenericJackson2JsonRedisSerializer 而非 Jackson2JsonRedisSerializer，因为前者保留类型信息。

### 4. HikariCP 连接池参数
- core/content-service: max-pool-size=20（一般业务量）
- file-service: max-pool-size=10（文件操作并发低）
- idle-timeout: 5min, max-lifetime: 30min, connection-timeout: 30s

### 5. Docker 健康检查
- MySQL: `mysqladmin ping` 每 10s，start_period 30s（等待初始化）
- Redis: `redis-cli ping` 每 10s
- 服务依赖使用 `condition: service_healthy` 确保基础设施就绪后再启动
