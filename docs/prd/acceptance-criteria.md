# 九、验收标准

## 9.1 功能验收

| 验收项 | 验收标准 |
|--------|----------|
| 前端页面 | 所有页面正常渲染、无报错、无明显样式问题 |
| 导航功能 | 导航栏点击跳转正确、搜索功能正常 |
| 新闻资讯 | 首页入口正常显示（导航栏不显示）；列表/详情页正常；无摘要字段 |
| 项目介绍 | 项目名称、项目内容正常显示；富文本内容渲染正常 |
| 内容展示 | 政策文件（不显示文号，含发文机构）、示范校共体（Tab切换）、资源共享（显示分类）内容正常展示 |
| 分页功能 | 默认10条、可选条数、显示总数、分页控件正常 |
| 独立列表/详情页 | 活动/月报独立列表页、详情页跳转正常 |
| 置顶功能 | 置顶内容优先显示 |
| 文件下载 | 文件可正常下载、预览 |
| 管理员登录 | 登录流程正常、首次强制改密、权限隔离正确；学校管理员无姓名字段 |
| 超管工作台 | 统计卡片显示正确；快捷操作入口正常 |
| 学校工作台 | 活动列表、月报列表显示正确；分页、编辑、删除、发布按钮正常 |
| 越权登录 | 超管可一键登录学校后台，操作记录日志 |
| 内容发布 | 支持图文编辑（富文本含首行缩进）、附件上传、外部链接三种发布方式 |
| 外部链接发布 | 外部链接方式：仅需填写标题和URL；列表显示"外链"标识；点击直接跳转外部页面 |
| 成员校管理 | 学校管理员可添加/编辑/删除/排序成员校；前端校共同体介绍页正确显示成员校列表 |
| 资源删除 | 超管可删除全站资源；学校管理员可删除本校资源 |
| 月报管理 | 超管可查看/删除学校月报；学校管理员可删除本校月报 |
| 示范校列表 | 列表不显示统计信息；不提供批量操作 |
| 状态筛选 | 新闻、政策、资源后台管理支持按发布状态筛选 |
| 敏感词过滤 | 发布时自动检测敏感词 |
| 数据备份 | 自动备份正常执行 |

## 9.2 性能验收

| 验收项 | 验收标准 |
|--------|----------|
| 页面加载 | 首页加载时间 ≤ 3秒 |
| 接口响应 | 常规接口响应 ≤ 500ms |
| 分页加载 | 列表分页加载 ≤ 500ms |
| 并发测试 | 500并发用户系统正常运行 |
| 下载测试 | 100并发下载系统正常运行 |

## 9.3 安全验收

| 验收项 | 验收标准 |
|--------|----------|
| HTTPS | 全站启用HTTPS |
| SQL注入 | 无SQL注入漏洞 |
| XSS攻击 | 无XSS漏洞 |
| 密码安全 | 密码 BCrypt 加密存储、符合复杂度要求（8-20位，大写+小写+数字）、首次强制改密 |
| 权限控制 | 无越权访问（越权登录功能除外，需记录日志） |

## 9.4 兼容性验收

| 验收项 | 验收标准 |
|--------|----------|
| Chrome | 80+版本正常运行 |
| Firefox | 75+版本正常运行 |
| Edge | 80+版本正常运行 |
| Safari | 13+版本正常运行 |
