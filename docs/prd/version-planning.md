# 八、版本规划

## 8.1 一期（P0）- 网站建设

**目标**：完成平台基础搭建与后台运营闭环

**周期**：[待定]

**范围**：

| 模块 | 功能 |
|------|------|
| 前端 | 首页（含新闻资讯入口）、项目介绍（项目名称+内容）、新闻资讯（导航不显示）、政策文件（含置顶、发文机构）、示范校共体（含Tab切换、独立列表/详情页、分页）、资源共享（含置顶）、全站搜索 |
| 超级管理后台 | 工作台（统计卡片、快捷操作）、用户管理（手机号注册、默认密码、无姓名字段、支持搜索筛选）、栏目管理（一级/二级）、内容管理（新闻资讯、政策文件、资源共享含状态筛选、项目介绍编辑、月报管理、置顶功能）、示范校管理（无统计无批量、含显示控制、越权登录）、系统设置（基础配置、权限设置、敏感词管理、数据备份） |
| 学校管理后台 | 工作台（活动列表、月报列表）、个人中心（首次改密）、栏目管理、内容管理（校共同体介绍、校共同体活动列表视图、校共同体月报列表视图含删除、富文本编辑器）、成员校管理 |

**交付物**：
- 可访问的官网前端
- 可操作的超级管理后台
- 可操作的学校管理后台
- 部署文档
- 操作手册

## 8.2 二期（P1）- 智能体

**目标**：完善用户体系、建设知识库、上线智能体服务，实现个性化服务与智能决策支持，平衡大模型使用成本，探索可持续运营模式

**周期**：[待定]

**技术选型**：
- 大模型：通义千问（阿里云）
- 知识库检索：向量数据库 + RAG（检索增强生成）模式
- 推荐引擎：基于用户身份标签 + 浏览行为的混合推荐

**范围**：

| 模块 | 功能 |
|------|------|
| 前端 — 注册登录 | 手机号/邮箱注册（含唯一身份标签选择，7种身份）、登录、密码找回 |
| 前端 — 个人中心 | 个人信息管理（身份不可修改）、浏览历史（近3个月）、收藏管理（永久保存） |
| 前端 — 智能体 | 智能问答（RAG 对话，引用知识来源）、个性化内容推荐（基于身份+行为）、数据可视化分析报告（面向教育行政/研究者）、个性化提醒（收藏更新/政策推送） |
| 超级管理后台 — 用户管理 | 普通用户列表（按身份/手机号筛选、导出）、账号管控（禁用/启用、智能体权限管控） |
| 超级管理后台 — 知识库管理 | 平台内容自动采集入库（政策/新闻/资源/校共体信息自动同步）、FAQ 手动录入（批量导入）、专家问答录入（富文本）、知识分类与标签管理、知识审核与发布、知识库统计仪表盘 |
| 超级管理后台 — 智能体管理 | 智能体配置（授权模式/使用额度/模型参数）、邀请码管理（批量生成/有效期/使用状态）、使用统计（日活/问答次数/满意度）、对话记录管理（质量监控） |
| 超级管理后台 — 运维 | 内容统计报表导出、操作日志管理 |
| 学校管理后台 | 内容查询（筛选本人发布内容）、本地资源管理（管理本校上传资源） |

**运营模式**：
- 智能体功能因大模型使用频次高、成本高，采用**收费使用**或**内部邀请码**模式
- 超管可在后台灵活切换授权模式，配置每日使用额度
- 未授权用户可看到功能入口但无法使用，引导获取邀请码或付费

**依赖**：
- 一期功能稳定运行
- 通义千问 API 接入就绪
- 向量数据库部署完成
- 用户身份标签体系设计完成
- 知识库内容初始化完成（平台已有内容采集 + 首批 FAQ 录入）
