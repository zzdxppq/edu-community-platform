# 二、用户分析

## 2.1 用户类型

| 用户类型 | 细分群体 | 一期权限 | 二期权限 |
|----------|----------|----------|----------|
| **访客** | 教师、教研员、家长、媒体、教育行政人员等 | 浏览所有公开内容 | 注册后使用智能体 |
| **学校管理员** | 各示范校专属管理员 | 管理本校内容 | 同一期 |
| **超级管理员** | 平台运营人员 | 全平台管理权限 | 新增普通用户管理 |

## 2.2 用户画像

### 画像1：访客用户

```
姓名：张老师
身份：乡镇小学语文教师
年龄：35岁
目标：了解城乡教育一体化政策，学习优秀学校经验
痛点：信息分散，难以找到权威的政策解读和实践案例
场景：工作间隙使用电脑浏览平台，查看政策文件、下载教学资源
```

### 画像2：学校管理员

```
姓名：李主任
身份：示范校教务主任，兼任平台管理员
年龄：42岁
目标：及时更新本校共同体建设动态，展示学校成果
痛点：需要简单易用的后台，不具备专业技术能力
场景：每月更新学校动态，上传月报，发布活动信息
```

### 画像3：超级管理员

```
姓名：王编辑
身份：平台运营人员
年龄：28岁
目标：高效管理平台内容，保障平台正常运行
痛点：需要统一管理多个学校的内容，及时处理不合规信息
场景：日常维护栏目、发布政策、管理学校管理员账号
```

## 2.3 使用场景

| 场景ID | 场景名称 | 用户 | 描述 | 优先级 |
|--------|----------|------|------|--------|
| UC-01 | 信息浏览 | 访客 | 浏览政策文件、示范校介绍、校共同体活动、资源共享等内容 | P0 |
| UC-02 | 资源下载 | 访客 | 在线预览并下载政策文件、教学资源 | P0 |
| UC-03 | 内容搜索 | 访客 | 通过关键词搜索政策、学校、活动、资源 | P0 |
| UC-04 | 后台运营 | 超级管理员 | 管理栏目、发布内容、维护示范校信息、管理学校管理员 | P0 |
| UC-05 | 学校管理 | 学校管理员 | 维护本校校共同体介绍、发布校共同体活动、上传月报、管理成员校 | P0 |
| UC-06 | 管理员登录 | 管理员 | 通过统一登录入口进入对应后台 | P0 |
| UC-07 | 用户注册 | 访客 | 注册账号，选择身份标签 | P1 |
| UC-08 | 智能问答 | 注册用户 | 使用智能体获取个性化服务 | P1 |
