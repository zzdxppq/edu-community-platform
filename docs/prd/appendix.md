## 附录A：术语表

| 术语 | 说明 |
|------|------|
| 示范校 | 参与城乡学校共同体建设的示范学校 |

| 校共体 | 学校共同体的简称 |
| 校共同体活动 | 示范校发布的建设动态、阶段成果、活动信息 |

| 工作台 | 后台首页，展示统计数据和快捷操作入口 |
| 底部快捷链接 | 原官网底部的项目介绍、政策文件、示范校共体、资源共享快捷入口（已删除） |
| 全站统一富文本编辑器 | 所有内容发布页面使用的统一富文本编辑器，详见3.5节功能规范 |
