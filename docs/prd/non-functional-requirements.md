# 四、非功能需求

## 4.1 性能需求

| 需求ID | 需求项 | 需求描述 | 指标 |
|--------|--------|----------|------|
| NFR-01 | 页面加载 | 首页及主要页面首次加载时间 | ≤ 3秒（100Mbps网络环境） |
| NFR-02 | 接口响应 | 常规接口响应时间 | ≤ 500ms |
| NFR-03 | 并发用户 | 系统支持同时在线用户数 | ≥ 500人 |
| NFR-04 | 下载带宽 | 服务器出口带宽 | ≥ 100Mbps |
| NFR-05 | 下载响应 | 文件下载首字节响应时间 | ≤ 3秒 |
| NFR-06 | 并发下载 | 支持同时下载用户数 | ≥ 100人 |
| NFR-07 | 分页加载 | 列表分页加载响应时间 | ≤ 500ms |

## 4.2 安全需求

| 需求ID | 需求项 | 需求描述 |
|--------|--------|----------|
| NFR-08 | 传输加密 | 全站启用HTTPS |
| NFR-09 | 密码安全 | 密码8-20位，必须包含大写字母、小写字母和数字，加密存储（BCrypt） |
| NFR-10 | 敏感词过滤 | 发布内容时自动检测敏感词 |
| NFR-11 | 外部链接安全 | 仅支持http/https协议，跳转前提示用户 |
| NFR-12 | 图片校验 | 支持jpg/png/gif格式，单张≤5MB |
| NFR-13 | SQL注入防护 | 防止SQL注入攻击 |
| NFR-14 | XSS防护 | 防止跨站脚本攻击 |

## 4.3 可靠性需求

| 需求ID | 需求项 | 需求描述 | 指标 |
|--------|--------|----------|------|
| NFR-15 | 系统可用性 | 系统正常运行时间占比 | ≥ 99.5% |
| NFR-16 | 数据备份 | 每日自动全量备份 | 凌晨2:00-4:00 |
| NFR-17 | 备份保留 | 备份文件保留期限 | 30天 |
| NFR-18 | 数据恢复 | 恢复时间目标（RTO） | ≤ 4小时 |
| NFR-19 | 备份监控 | 备份失败时自动告警 | 实时通知 |

## 4.4 兼容性需求

| 需求ID | 需求项 | 需求描述 |
|--------|--------|----------|
| NFR-20 | 浏览器兼容 | Chrome 80+、Firefox 75+、Edge 80+、Safari 13+ |
| NFR-21 | 分辨率 | 支持1280×720及以上分辨率 |
| NFR-22 | 移动端 | 一期暂不支持，二期评估 |

## 4.5 文件规格

| 文件类型 | 支持格式 | 大小限制 |
|----------|----------|----------|
| 图片 | jpg/png/gif | 单张≤5MB |
| 文档 | pdf/doc/docx/xls/xlsx | 单文件≤50MB |
| 视频 | mp4/mov | 单文件≤200MB |
