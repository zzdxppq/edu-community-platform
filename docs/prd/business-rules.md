# 六、业务规则

## 6.1 权限规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-01 | 开放浏览 | 前端核心内容无需登录即可浏览，仅智能体功能需登录 |
| BR-02 | 权限隔离 | 学校管理员仅可操作本校内容，无平台级权限 |
| BR-03 | 超级权限 | 超级管理员拥有全平台权限，可直接删除不合规内容 |
| BR-04 | 账号关联 | 每个学校管理员账号必须关联一所示范校 |
| BR-05 | 多账号 | 每所示范校可创建多个管理员账号 |

## 6.2 账号管理规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-AC-01 | 账号格式 | 管理员账号**必须为手机号**（11位数字） |
| BR-AC-02 | 默认密码 | 新增管理员时无需设置密码，默认密码为**手机号后6位** |
| BR-AC-03 | 首次改密 | 首次登录**强制修改密码**，修改后方可进入后台 |
| BR-AC-04 | 密码重置 | 重置密码后恢复为手机号后6位，并标记为需首次改密状态 |
| BR-AC-05 | 密码规则 | 密码8-20位，必须包含大写字母、小写字母和数字 |

## 6.3 栏目管理规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-CM-01 | 一级栏目 | 超级管理员可管理前端导航一级栏目，支持新增、隐藏、修改、删除、排序 |
| BR-CM-02 | 二级菜单启用 | 超级管理员可设置一级栏目是否启用二级菜单（开关控制） |
| BR-CM-03 | 二级菜单维护 | 启用二级菜单后，可新增、编辑、删除、排序二级菜单项 |
| BR-CM-04 | 二级菜单关联 | 二级菜单需关联对应的内容分类，用于前端筛选展示 |
| BR-CM-05 | 示范校特殊处理 | 示范校共体栏目的二级内容（校共同体介绍、校共同体活动、校共同体月报）通过Tab切换实现，不在左侧二级菜单显示 |
| BR-CM-06 | 栏目保护 | 系统预置栏目（首页、项目介绍、政策文件、示范校共体、资源共享、登录）不可删除，仅支持隐藏 |

## 6.4 内容发布规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-06 | 发布方式 | 支持三种方式：①图文编辑（富文本编辑器）；②附件上传（可选，支持PDF/Word等）；③外部链接（仅填写标题和URL，点击后直接跳转外部页面） |
| BR-06-1 | 外部链接发布 | 选择"外部链接"方式时，仅需填写：标题、外部URL、封面图（可选）、摘要（可选） |
| BR-06-2 | 外部链接展示 | 外部链接类型内容在列表中显示"外链"标识，点击后直接跳转至外部页面（非进入详情页） |
| BR-07 | 无需审核 | 学校管理员发布内容直接上线，无需审核 |
| BR-08 | 首页同步 | 学校管理员发布"校共同体活动"后自动同步至首页动态区块 |
| BR-09 | 来源标识 | 同步的动态自动标注发布学校名称 |
| BR-10 | 敏感词 | 发布内容时自动检测敏感词，命中时提示修改 |
| BR-10-1 | 项目介绍编辑 | 项目介绍为单页内容，使用富文本编辑器编辑，支持图片、视频、表格；由超级管理员维护 |

## 6.5 内容排序与置顶规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-TP-01 | 默认排序 | 政策文件、资源共享、活动、月报默认按发布时间**倒序**排列（最新在前） |
| BR-TP-02 | 置顶功能 | 政策文件、资源共享内容支持设置**置顶**，置顶内容优先显示 |
| BR-TP-03 | 置顶排序 | 多个置顶内容之间按发布时间倒序排列 |
| BR-TP-04 | 首页展示 | 首页各模块入口显示最新5条，置顶内容优先 |

## 6.6 示范校管理规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-11 | 基础信息 | 学校名称、学段、所属区域（精确到县区）、联系电话由超级管理员维护 |
| BR-12 | 详情内容 | 共同体组建模式、核心优势、详细介绍由学校管理员使用富文本编辑器维护；不需要添加成员校列表 |
| BR-13 | 官网显示控制 | 超级管理员可设置示范校是否在官网显示、显示排序 |
| BR-14 | 官网默认显示 | 官网示范校导航默认显示前**6个**（按排序），支持"查看全部"展开 |
| BR-15 | 越权登录 | 超级管理员可从后台一键登录任意学校管理后台（模拟登录），用于运维排查；操作需记录日志 |
| BR-16 | 快速访问 | 超级管理员可从后台快速跳转至示范校前端主页 |

## 6.7 成员校管理规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-MB-01 | 管理权限 | 成员校由**学校管理员**维护，超级管理员可查看 |
| BR-MB-02 | 前端展示 | 成员校信息展示到前端，在校共同体介绍页面显示**成员校列表**（学校名称） |
| BR-MB-03 | 展示内容 | 前端展示：成员校数量统计（如"共有成员校 12 所"）+ 成员校名称列表 |
| BR-MB-04 | 信息字段 | 成员校信息包括：学校名称、学校代码（可选）、排序 |
| BR-MB-05 | 排序规则 | 成员校按排序字段升序展示，支持学校管理员调整顺序 |

## 6.8 外部链接规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-17 | 协议限制 | 仅支持http/https协议，禁止javascript等协议 |
| BR-18 | 跳转提示 | 点击外部链接时弹出提示"您即将离开本平台" |
| BR-19 | 链接标识 | 详情页中外部链接以独立样式展示，明确标识 |
| BR-20 | 白名单 | 可配置信任域名列表，白名单内链接不弹提示 |

## 6.9 内容删除规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-21 | 超管删除 | 超级管理员可直接删除不合规内容（含学校月报、**全站资源共享**），无需审核 |
| BR-22 | 学校删除月报 | 学校管理员可删除本校发布的月报 |
| BR-22-1 | 学校删除资源 | 学校管理员可删除**本校上传的资源**（仅限本校资源） |
| BR-23 | 删除记录 | 删除后保留删除记录（删除人、时间、原内容摘要） |
| BR-24 | 前端不可见 | 删除后前端立即不可见 |
| BR-25 | 栏目保护 | 栏目下已有数据不能删除只能隐藏，无数据可删除 |

## 6.10 分页规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-PG-01 | 默认条数 | 所有列表页默认每页显示**10条** |
| BR-PG-02 | 可选条数 | 支持选择每页显示条数：10/20/50/100 |
| BR-PG-03 | 总数显示 | 列表底部显示总条数，如"共 128 条记录" |
| BR-PG-04 | 独立列表页 | 活动、月报点击"查看更多"进入独立列表页面（非同页加载），避免数据过多卡顿 |

## 6.11 数据备份规则

| 规则ID | 规则名称 | 规则描述 |
|--------|----------|----------|
| BR-26 | 自动备份 | 每日凌晨2:00-4:00自动全量备份 |
| BR-27 | 备份内容 | 包括数据库数据、上传文件（图片、附件） |
| BR-28 | 保留期限 | 保留最近30天备份，超期自动清理 |
| BR-29 | 物理隔离 | 备份文件存储于独立服务器/云存储 |
| BR-30 | 手动备份 | 支持手动触发备份，每日限1次 |
