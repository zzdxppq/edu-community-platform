**页面URL规则：**
- 学校主页：`/schools/:id`
- 活动列表：`/schools/:id/activities`
- 活动详情：`/schools/:id/activities/:activityId`
- 月报列表：`/schools/:id/reports`
- 月报详情：`/schools/:id/reports/:reportId`

**视觉风格说明：**
- 页面背景：使用学校实景图片作为背景（替代纯蓝色背景），展示学校风貌
- 背景图片规格：建议尺寸1920×400px，支持jpg/png格式，单张≤2MB
- 图片效果：背景图片可添加适度遮罩层以保证文字可读性
- 默认背景：未上传背景图的学校使用平台默认图片

## 3.5 富文本编辑器功能规范

| 功能分类 | 功能项 | 说明 |
|----------|--------|------|
| 基础格式 | 加粗、斜体、下划线、删除线 | 文字基础样式 |

| 字体设置 | 字体颜色 | 支持自定义颜色选择 |
| 字体设置 | 背景色 | 文字背景高亮色 |

| 段落格式 | **首行缩进** | 支持设置首行缩进（2字符） |
| 段落格式 | 行间距 | 支持1.0/1.5/2.0/2.5/3.0倍行距 |
| 段落格式 | 段落间距 | 段前、段后间距设置 |
| 列表功能 | 有序列表 | 数字编号列表 |
| 列表功能 | 无序列表 | 符号列表 |
| 插入功能 | 图片 | 支持上传图片，可设置大小、对齐 |
| 插入功能 | 视频 | 支持上传视频或嵌入外部视频链接 |
| 插入功能 | 表格 | 支持插入表格，可编辑行列 |
| 插入功能 | 链接 | 支持插入超链接 |
| 插入功能 | 分割线 | 水平分割线 |
| 其他功能 | 撤销/重做 | 操作回退 |
